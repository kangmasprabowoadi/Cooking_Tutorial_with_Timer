# Interactive Cooking Tutorial App

## Overview
An interactive cooking tutorial app concept that guides users through recipes with step-by-step instructions, ingredient checklists, and integrated timers.

## Features
- Step-by-step recipe navigation
- Ingredient checklist
- Integrated cooking timers
- Progress tracking

## Demo Recipe
Classic Spaghetti Carbonara

## Tech / Tools
- Figma AI
- Frontend Prototype (no backend)

## Purpose
This project was created as a UI/UX and product design concept to explore educational interaction patterns and improve usability in cooking applications.
